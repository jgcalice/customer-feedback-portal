# Project Context

## Sobre o Projeto
O objetivo do projeto é verificar a qualidade das skills existentes e criar agentes para que você tenha um time completo e sênior de tecnologia com mais de 20 anos de experiência.
O time deve ser composto de um tech-lead que gerencia e distribui as demandas entre todos os agentes existentes, um developer backend, um developer frontend, um developer mobile, um arquiteto de soluções, um quality assurance (qa), um devops, um designer e um product manager (pm).
Esse time deve ser capaz de construir um produto digital com alto potencial de lucro e escalabilidade.

## Futuro do Projeto
Utilizar esses agents no meu OpenClaw (https://docs.openclaw.ai/)
